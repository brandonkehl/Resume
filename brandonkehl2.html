About This Website

This website was created to demonstrate various AWS concepts and gain hands on experience with those concepts.

  • This website is deployed on S3.
  • I created a CloudFront distribution to require HTTPS to access this website.
  • I purchased a custom domain name with Route 53 and pointed it to my new CloudFront distribution.
  • The visitor counter (made from Javascript) points to a DynamoDB table. The counter retrieves and updates in a DynamoDB database. To do this I had to create an API Gateway and a Lambda function to create, delete, retrieve items from the DynamoDb table.
  • I utilized CodePipeline to sync with my GitHub repository as a demonstration of CI/CD.
  • There will be more to come!
